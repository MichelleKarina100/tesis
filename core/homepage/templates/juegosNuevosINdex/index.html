Matemáticas: Conjuntos






Concepto:

En matemáticas, un conjunto es una colección de elementos considerada en sí misma como un objeto. Los elementos de un conjunto, pueden ser las siguientes: personas, números, colores, letras, figuras, etc. Se dice que un elemento pertenece al conjunto si está definido como incluido de algún modo dentro de él.

1Conjunto:

Podemos afirmar que un conjunto es una colección de cosas, personas, objetos, etc.

2Elemento:

Cada una de las cosas que forman parte del conjunto recibe le nombre de elemento.

3Nombre del Conjunto:

Todo conjunto tiene un nombre con el cual lo vamos a identificar con letras mayúsculas A, B, C, D, etc.

Tipos de Conjuntos:



Conjunto vacío

El conjunto vacío es aquel que no tiene elementos. En matemáticas, el conjunto vacío es el conjunto que carece de elementos.

Conjunto Unitario

El conjunto unitario se distingue por tener solo un elemento. No importa qué tipo de elemento tenga el conjunto, un gato, un perro, un número, etc.

Conjunto universo

El conjunto universo es el que esta formado por todos los elementos del tema tratado.

Ejemplo: Este conjunto universo tiene dos subconjuntos que son: frutas y verduras.

Conjunto finito

Un conjunto finito, en otras palabras, es aquel que posee un número de elementos que pueden contarse. Siendo lo opuesto a un conjunto infinito, donde los elementos son incontables.

Conjunto infinito

Los conjuntos infinitos son aquellos a los cuales no podemos contar la cantidad de elementos que los componen.

Matamáticas Kilogramo y Gramo






Concepto:

Las unidades de medida son una referencia que se utiliza para medir o calcular asuntos como la longitud, el peso la temperatura, el tiempo , la capacidad y muchas otras.

1Kilogramo:

Para medir el peso de un objeto, persona o animal utilizamos el kilogramo, también llamado kilo.Se representa con las letras "kg".

2Gramo:

Para medir el peso de objetos, pequeños utilizamos el gramo, se representa con la letra "g".

Kilogramo y Gramo:


"La masa es una magnitud que expresa la cantidad de materia que tiene un cuerpo. Se mide en Kilogramos "Kg" y se calcula con balanzas y básculas"
